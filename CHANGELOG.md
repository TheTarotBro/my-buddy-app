# My Circle — Changelog

## v2.0 — April 6, 2026
### Places + Cross-Linked Events
- **Places tab**: New 4th tab for tracking places — name, rating (1-5 stars), notes, Google Maps link (opens in Maps app on iOS)
- **Visit logging**: Log visits on places with date, note, and tag people who were there
- **Cross-linked events**: Logging a visit with tagged people auto-creates "In Person" touchpoints on each person's profile. Deleting a visit removes all linked touchpoints. Each person's touchpoint shows the linked place name
- **"Coffee" → "In Person"**: Touchpoint type renamed to support place-linked events
- **Shared event IDs**: Visits and touchpoints are connected by event ID for data integrity
